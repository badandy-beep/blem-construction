# Michael Blem Construction - Website v5.0.0

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Connect repo to Vercel
3. Deploy

## Tech Stack

- Next.js 14.2.3
- React 18.3.1
- Tailwind CSS 3.4.3
- Framer Motion 11.0.0
